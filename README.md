# 🐹 AIT HAMSTER

Телеграм веб-клиер игра с хомячком! Кликай, зарабатывай монеты и улучшайся! 

## 🎮 Особенности

- ✅ **Динамичный клиер** - Кликай по хомячку и зарабатывай монеты
- ✅ **Система энергии** - Управляй энергией, восстанавливается со временем
- ✅ **Апгрейды и улучшения** - 4 типа апгрейдов для развития
- ✅ **Пассивный доход** - Получай монеты автоматически
- ✅ **Уровни и прогресс** - Следи за своим развитием
- ✅ **Сохранение прогресса** - Данные сохраняются локально
- ✅ **Красивый UI** - Оптимизированный интерфейс для мобильных
- ✅ **Telegram интеграция** - Полная поддержка Telegram Web App

## 🚀 Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Шаги установки

```bash
# 1. Клонируем репозиторий
git clone https://github.com/s71503394-cmd/ait-hamster.git
cd ait-hamster

# 2. Устанавливаем зависимости
npm install

# 3. Запускаем в режиме разработки
npm run dev

# 4. Собираем для продакшена
npm run build
```

## 🎯 Как играть

1. **Кликай по хомячку** - каждый клик даёт монеты
2. **Следи за энергией** - энергия расходуется и восстанавливается
3. **Покупай апгрейды** - улучшай клик, энергию, пассивный доход
4. **Достигай уровней** - чем больше монет, тем выше уровень
5. **Получай пассивный доход** - развивайся ещё пока не играешь

## 💡 Механики игры

### Система энергии
- Каждый клик стоит 1 энергию
- Энергия восстанавливается со скоростью ~10 энергии/сек
- Можно купить апгрейд для увеличения максимума энергии
- Можно ускорить регенерацию через апгрейды

### Монеты
- Базовый доход: 1 монета за клик
- Пассивный доход: увеличивается через апгрейды
- Всё сохраняется в localStorage

### Уровни
- Уровень = (всего монет / 10000) + 1
- Отслеживай прогресс в реальном времени

## 🔧 Конфигурация игры

Отредактируй значения в `src/services/gameStore.ts`:

```typescript
const DEFAULT_CONFIG: GameConfig = {
  initialCoins: 0,          // Начальные монеты
  initialEnergy: 1000,      // Начальная энергия
  maxEnergy: 1000,          // Максимум энергии
  energyCostPerClick: 1,    // Энергия на клик
  coinPerClick: 1,          // Монеты за клик
  energyRegenRate: 10,      // Регенерация энергии/сек
  energyRegenDelay: 1000,   // Задержка регенерации (мс)
};
```

## 🐛 Известные проблемы и решения

### Энергия не восстанавливается
- Проверь что hook `useEnergyRegen` инициализирован
- Убедись что интервал не очищен слишком рано

### Данные не сохраняются
- localStorage должна быть доступна
- Проверь что браузер не в приватном режиме
